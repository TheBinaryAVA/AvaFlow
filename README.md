# 🌊 AvaFlow – Personal Productivity & Task Tracker

**AvaFlow** is a modern web-based personal productivity application built with **HTML, CSS, and JavaScript**. It helps users manage their tasks, track completion streaks, and stay motivated with a visually appealing dashboard and interactive timers.

---

## ✨ Key Features

- **Add, edit, and delete tasks** with deadlines    
- **Track daily streaks** for completed tasks  
- **Dashboard view** with task statistics and progress  
- **Responsive and clean UI** for easy usability  
- **Persistent data storage** using localStorage  
- Sync tasks with a backend database  
- Dark/light mode toggle  
- Notifications and reminders  
- Analytics charts for productivity  
---

## 🛠 Tech Stack

- **HTML5** – Structure and layout  
- **CSS3** – Styling, animations, and responsive design  
- **JavaScript** – Task management, timers, streak calculations, and DOM manipulation  
- **LocalStorage** – For saving user tasks and streak data  

---

## 🚀 How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/avaflow.git
```

2. Navigate into the project folder:

```bash
cd avaflow
```

3. Open `index.html` in your preferred web browser.

No installation required — fully front-end web app.
